node-randomstring
=================

Random String generator for node.js

## Usage

```javascript
var randomstring = require("foostring");

randomstring.generate();
// >> "XwPp9xazJ0ku5CZnlmgAx2Dld8SHkAeT"

randomstring.generate(7);
// >> "xqm5wXX"
```
